# 2.3 前端组件生成

> 从设计稿到可复用组件，让 UI 开发效率翻倍。

## 🎯 场景描述

你需要实现一个复杂的前端组件，传统流程：

1. 分析设计稿，拆解组件结构
2. 搭建 HTML 骨架
3. 写 CSS 样式（布局、颜色、间距）
4. 加交互逻辑（状态、事件）
5. 调试响应式、边界情况

**预计耗时：2-4 小时**

大部分时间花在：
- 调 CSS 对齐
- 写重复的状态管理
- 处理 edge case

## ✨ Claude Code 做法

直接描述你要的组件：

```
帮我写一个 React 文件上传组件：

功能：
1. 支持拖拽上传和点击上传
2. 显示上传进度条
3. 支持多文件上传
4. 显示已上传文件列表，可删除
5. 文件类型限制：图片（jpg/png/gif）和 PDF
6. 单个文件最大 10MB

UI 要求：
- 使用 Tailwind CSS
- 拖拽区域要有视觉反馈（悬停、拖拽中状态）
- 进度条用渐变色
- 错误提示用红色边框+文字
- 整体风格简洁现代

交互细节：
- 拖拽进入时区域高亮
- 上传中显示 loading 动画
- 上传成功显示绿色勾选
- 上传失败可重试

返回完整的组件代码，包含 TypeScript 类型定义。
```

```tsx
// FileUploader.tsx
import React, { useState, useRef, useCallback } from 'react';

interface FileItem {
  id: string;
  file: File;
  name: string;
  size: number;
  progress: number;
  status: 'pending' | 'uploading' | 'success' | 'error';
  error?: string;
}

interface FileUploaderProps {
  onUpload?: (files: File[]) => Promise<void>;
  onRemove?: (file: FileItem) => void;
  maxSize?: number; // MB
  accept?: string;
}

const FileUploader: React.FC<FileUploaderProps> = ({
  onUpload,
  onRemove,
  maxSize = 10,
  accept = '.jpg,.jpeg,.png,.gif,.pdf'
}) => {
  const [files, setFiles] = useState<FileItem[]>([]);
  const [isDragging, setIsDragging] = useState(false);
  const inputRef = useRef<HTMLInputElement>(null);

  const generateId = () => Math.random().toString(36).substr(2, 9);

  const validateFile = (file: File): string | null => {
    const validTypes = ['image/jpeg', 'image/png', 'image/gif', 'application/pdf'];
    if (!validTypes.includes(file.type)) {
      return '只支持 JPG、PNG、GIF 和 PDF 文件';
    }
    if (file.size > maxSize * 1024 * 1024) {
      return `文件大小不能超过 ${maxSize}MB`;
    }
    return null;
  };

  const addFiles = (newFiles: FileList | null) => {
    if (!newFiles) return;

    const newFileItems: FileItem[] = [];
    
    Array.from(newFiles).forEach(file => {
      const error = validateFile(file);
      newFileItems.push({
        id: generateId(),
        file,
        name: file.name,
        size: file.size,
        progress: 0,
        status: error ? 'error' : 'pending',
        error: error || undefined
      });
    });

    setFiles(prev => [...prev, ...newFileItems]);

    // 自动开始上传非错误文件
    newFileItems.forEach(item => {
      if (!item.error) {
        simulateUpload(item.id);
      }
    });
  };

  const simulateUpload = async (fileId: string) => {
    setFiles(prev => prev.map(f => 
      f.id === fileId ? { ...f, status: 'uploading' } : f
    ));

    // 模拟上传进度
    const interval = setInterval(() => {
      setFiles(prev => {
        const file = prev.find(f => f.id === fileId);
        if (!file || file.status !== 'uploading') {
          clearInterval(interval);
          return prev;
        }
        
        const newProgress = Math.min(file.progress + 10, 100);
        if (newProgress === 100) {
          clearInterval(interval);
          setTimeout(() => {
            setFiles(p => p.map(f => 
              f.id === fileId ? { ...f, status: 'success', progress: 100 } : f
            ));
          }, 200);
        }
        
        return prev.map(f => 
          f.id === fileId ? { ...f, progress: newProgress } : f
        );
      });
    }, 100);
  };

  const removeFile = (id: string) => {
    const file = files.find(f => f.id === id);
    if (file && onRemove) {
      onRemove(file);
    }
    setFiles(prev => prev.filter(f => f.id !== id));
  };

  const retryUpload = (id: string) => {
    setFiles(prev => prev.map(f => 
      f.id === id ? { ...f, status: 'pending', error: undefined, progress: 0 } : f
    ));
    simulateUpload(id);
  };

  const handleDragEnter = useCallback((e: React.DragEvent) => {
    e.preventDefault();
    setIsDragging(true);
  }, []);

  const handleDragLeave = useCallback((e: React.DragEvent) => {
    e.preventDefault();
    setIsDragging(false);
  }, []);

  const handleDrop = useCallback((e: React.DragEvent) => {
    e.preventDefault();
    setIsDragging(false);
    addFiles(e.dataTransfer.files);
  }, []);

  const formatSize = (bytes: number): string => {
    if (bytes < 1024) return bytes + ' B';
    if (bytes < 1024 * 1024) return (bytes / 1024).toFixed(1) + ' KB';
    return (bytes / (1024 * 1024)).toFixed(1) + ' MB';
  };

  return (
    <div className="w-full max-w-2xl mx-auto p-6">
      {/* 拖拽上传区域 */}
      <div
        onClick={() => inputRef.current?.click()}
        onDragEnter={handleDragEnter}
        onDragOver={handleDragEnter}
        onDragLeave={handleDragLeave}
        onDrop={handleDrop}
        className={`
          relative border-2 border-dashed rounded-xl p-8 text-center cursor-pointer
          transition-all duration-200 ease-in-out
          ${isDragging 
            ? 'border-blue-500 bg-blue-50' 
            : 'border-gray-300 hover:border-gray-400 hover:bg-gray-50'
          }
        `}
      >
        <input
          ref={inputRef}
          type="file"
          multiple
          accept={accept}
          onChange={(e) => addFiles(e.target.files)}
          className="hidden"
        />
        
        <div className={`
          mx-auto w-16 h-16 mb-4 rounded-full flex items-center justify-center
          transition-colors duration-200
          ${isDragging ? 'bg-blue-100' : 'bg-gray-100'}
        `}>
          <svg 
            className={`w-8 h-8 ${isDragging ? 'text-blue-600' : 'text-gray-500'}`}
            fill="none" 
            stroke="currentColor" 
            viewBox="0 0 24 24"
          >
            <path 
              strokeLinecap="round" 
              strokeLinejoin="round" 
              strokeWidth={2} 
              d="M7 16a4 4 0 01-.88-7.903A5 5 0 1115.9 6L16 6a5 5 0 011 9.9M15 13l-3-3m0 0l-3 3m3-3v12" 
            />
          </svg>
        </div>
        
        <p className="text-lg font-medium text-gray-700 mb-1">
          {isDragging ? '松开以上传文件' : '点击或拖拽文件到此处'}
        </p>
        <p className="text-sm text-gray-500">
          支持 JPG、PNG、GIF、PDF，单个文件最大 {maxSize}MB
        </p>
      </div>

      {/* 文件列表 */}
      {files.length > 0 && (
        <div className="mt-6 space-y-3">
          {files.map(file => (
            <div 
              key={file.id}
              className={`
                flex items-center p-4 rounded-lg border transition-all
                ${file.status === 'error' 
                  ? 'border-red-300 bg-red-50' 
                  : 'border-gray-200 bg-white'
                }
              `}
            >
              {/* 文件图标 */}
              <div className={`
                w-10 h-10 rounded-lg flex items-center justify-center flex-shrink-0
                ${file.status === 'error' ? 'bg-red-100' : 'bg-gray-100'}
              `}>
                {file.status === 'success' ? (
                  <svg className="w-5 h-5 text-green-500" fill="currentColor" viewBox="0 0 20 20">
                    <path fillRule="evenodd" d="M16.707 5.293a1 1 0 010 1.414l-8 8a1 1 0 01-1.414 0l-4-4a1 1 0 011.414-1.414L8 12.586l7.293-7.293a1 1 0 011.414 0z" clipRule="evenodd" />
                  </svg>
                ) : file.status === 'error' ? (
                  <svg className="w-5 h-5 text-red-500" fill="currentColor" viewBox="0 0 20 20">
                    <path fillRule="evenodd" d="M4.293 4.293a1 1 0 011.414 0L10 8.586l4.293-4.293a1 1 0 111.414 1.414L11.414 10l4.293 4.293a1 1 0 01-1.414 1.414L10 11.414l-4.293 4.293a1 1 0 01-1.414-1.414L8.586 10 4.293 5.707a1 1 0 010-1.414z" clipRule="evenodd" />
                  </svg>
                ) : (
                  <svg className="w-5 h-5 text-gray-500" fill="currentColor" viewBox="0 0 20 20">
                    <path fillRule="evenodd" d="M4 4a2 2 0 012-2h4.586A2 2 0 0112 2.586L15.414 6A2 2 0 0116 7.414V16a2 2 0 01-2 2H6a2 2 0 01-2-2V4z" clipRule="evenodd" />
                  </svg>
                )}
              </div>

              {/* 文件信息 */}
              <div className="ml-3 flex-1 min-w-0">
                <p className="text-sm font-medium text-gray-900 truncate">
                  {file.name}
                </p>
                <p className="text-xs text-gray-500">
                  {formatSize(file.size)}
                </p>
                
                {/* 进度条或错误信息 */}
                {file.status === 'uploading' && (
                  <div className="mt-2">
                    <div className="h-1.5 bg-gray-200 rounded-full overflow-hidden">
                      <div 
                        className="h-full bg-gradient-to-r from-blue-500 to-blue-600 rounded-full transition-all duration-200"
                        style={{ width: `${file.progress}%` }}
                      />
                    </div>
                    <p className="text-xs text-gray-500 mt-1">{file.progress}%</p>
                  </div>
                )}
                
                {file.error && (
                  <p className="text-xs text-red-600 mt-1">{file.error}</p>
                )}
              </div>

              {/* 操作按钮 */}
              <div className="ml-4 flex items-center space-x-2">
                {file.status === 'error' && (
                  <button
                    onClick={() => retryUpload(file.id)}
                    className="p-1.5 text-gray-500 hover:text-blue-600 hover:bg-blue-50 rounded-lg transition-colors"
                    title="重试"
                  >
                    <svg className="w-4 h-4" fill="none" stroke="currentColor" viewBox="0 0 24 24">
                      <path strokeLinecap="round" strokeLinejoin="round" strokeWidth={2} d="M4 4v5h.582m15.356 2A8.001 8.001 0 004.582 9m0 0H9m11 11v-5h-.581m0 0a8.003 8.003 0 01-15.357-2m15.357 2H15" />
                    </svg>
                  </button>
                )}
                
                <button
                  onClick={() => removeFile(file.id)}
                  className="p-1.5 text-gray-500 hover:text-red-600 hover:bg-red-50 rounded-lg transition-colors"
                  title="删除"
                >
                  <svg className="w-4 h-4" fill="none" stroke="currentColor" viewBox="0 0 24 24">
                    <path strokeLinecap="round" strokeLinejoin="round" strokeWidth={2} d="M19 7l-.867 12.142A2 2 0 0116.138 21H7.862a2 2 0 01-1.995-1.858L5 7m5 4v6m4-6v6m1-10V4a1 1 0 00-1-1h-4a1 1 0 00-1 1v3M4 7h16" />
                  </svg>
                </button>
              </div>
            </div>
          ))}
        </div>
      )}
    </div>
  );
};

export default FileUploader;
```

## 💡 写好组件的技巧

### 1. 先描述功能，再描述外观

```
# ✅ 好的描述
组件要支持多文件上传，每个文件显示进度条，
上传完成显示成功状态，失败可以重试。

# ❌ 不好的描述
做一个蓝色的上传按钮，上面有个云图标...
```

### 2. 说明交互状态

```
需要处理的状态：
- 默认状态
- 拖拽悬停状态
- 上传中状态
- 成功状态
- 错误状态
```

### 3. 指定技术栈

```
技术栈：
- React 18 + TypeScript
- Tailwind CSS
- 不使用第三方 UI 库
```

### 4. 要类型定义

```
返回完整的 TypeScript 类型定义，
包括 Props 和内部状态的类型。
```

## ⚠️ 注意事项

1. **检查生成的依赖**——确认没有引入不需要的库
2. ** accessibility 要自己补**——AI 生成的组件可能没有考虑无障碍
3. **性能优化要手动加**——大列表可能需要虚拟滚动
4. **样式要适配你的设计系统**——颜色、圆角可能需要调整

## 📝 小结

前端组件开发是重复性很高的工作——同样的布局、同样的交互模式、同样的状态管理。Claude Code 让你：

- 不用写重复的 useState/useCallback
- 不用调 CSS 对齐
- 不用处理繁琐的 edge case
- 专注于业务逻辑和用户体验

**好的组件设计在于清晰的接口，而不是代码量。**
