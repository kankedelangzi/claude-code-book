# Claude Code 案例指南

> 不同于官方文档的理论介绍，本书专注于**实战案例**，带你从零到精通。

## 📖 关于本书

- **目标读者**：想用 Claude Code 提升编程效率的开发者
- **特色**：真实案例 + 代码对比 + 效率对比
- **更新频率**：持续迭代，跟随 Claude Code 版本更新

## 📚 目录结构

```
01-快速入门案例/
02-日常开发场景/
03-重构与优化/
04-调试与排错/
05-项目实战/
06-进阶技巧/
07-常见问题FAQ/
```

## 🎯 案例风格

每个案例包含：
- 场景描述：什么情况下用
- 操作对比：传统做法 vs Claude Code 做法
- 效率提升：时间节省多少
- 注意事项：什么情况要小心

## 📁 资料存放

- `materials/` - PDF、参考文档等素材
- `images/` - 截图、流程图等图片
