### 案例 1：测试异步函数

**函数代码：**

```javascript
async function fetchUserWithRetry(userId, maxRetries = 3) {
  for (let i = 0; i < maxRetries; i++) {
    try {
      const response = await fetch(`/api/users/${userId}`)
      if (response.ok) {
        return await response.json()
      }
    } catch (error) {
      if (i === maxRetries - 1) throw error
      await new Promise(r => setTimeout(r, 1000 * (i + 1)))
    }
  }
  throw new Error('Max retries exceeded')
}
```

**让 Claude Code 生成测试：**

```
帮我为这个重试函数写 Jest 测试，需要 mock fetch API：

覆盖场景：
1. 第一次请求成功
2. 第一次失败，第二次成功
3. 全部失败，抛出错误
4. 返回 404 的情况
```

**生成的测试：**

```javascript
describe('fetchUserWithRetry', () => {
  beforeEach(() => {
    jest.clearAllMocks()
    global.fetch = jest.fn()
  })

  it('第一次请求成功时直接返回', async () => {
    global.fetch.mockResolvedValueOnce({
      ok: true,
      json: async () => ({ id: 1, name: 'Alice' })
    })

    const result = await fetchUserWithRetry(1)
    expect(result).toEqual({ id: 1, name: 'Alice' })
    expect(global.fetch).toHaveBeenCalledTimes(1)
  })

  it('第一次失败第二次成功时重试', async () => {
    global.fetch
      .mockRejectedValueOnce(new Error('Network error'))
      .mockResolvedValueOnce({
        ok: true,
        json: async () => ({ id: 1, name: 'Alice' })
      })

    const result = await fetchUserWithRetry(1)
    expect(result).toEqual({ id: 1, name: 'Alice' })
    expect(global.fetch).toHaveBeenCalledTimes(2)
  })

  it('全部失败时抛出错误', async () => {
    global.fetch.mockRejectedValue(new Error('Network error'))

    await expect(fetchUserWithRetry(1, 2)).rejects.toThrow('Network error')
    expect(global.fetch).toHaveBeenCalledTimes(2)
  })

  it('返回 404 时继续重试', async () => {
    global.fetch
      .mockResolvedValueOnce({ ok: false, status: 404 })
      .mockResolvedValueOnce({
        ok: true,
        json: async () => ({ id: 1, name: 'Alice' })
      })

    const result = await fetchUserWithRetry(1)
    expect(result).toEqual({ id: 1, name: 'Alice' })
  })
})
```

### 案例 2：测试 React 组件

**组件代码：**

```tsx
function UserForm({ onSubmit }) {
  const [name, setName] = useState('')
  const [email, setEmail] = useState('')

  const handleSubmit = (e) => {
    e.preventDefault()
    if (!name || !email) return
    onSubmit({ name, email })
  }

  return (
    <form onSubmit={handleSubmit}>
      <input 
        placeholder="Name" 
        value={name} 
        onChange={e => setName(e.target.value)} 
      />
      <input 
        placeholder="Email" 
        value={email} 
        onChange={e => setEmail(e.target.value)} 
      />
      <button type="submit">Submit</button>
    </form>
  )
}
```

**生成测试：**

```
用 React Testing Library 为这个表单组件写测试：

覆盖场景：
1. 正常提交
2. 空字段时不提交
3. 输入更新显示
```

**Claude Code 生成的测试：**

```tsx
import { render, screen, fireEvent } from '@testing-library/react'
import UserForm from './UserForm'

describe('UserForm', () => {
  it('正常提交时调用 onSubmit', () => {
    const onSubmit = jest.fn()
    render(<UserForm onSubmit={onSubmit} />)

    fireEvent.change(screen.getByPlaceholderText('Name'), {
      target: { value: 'Alice' }
    })
    fireEvent.change(screen.getByPlaceholderText('Email'), {
      target: { value: 'alice@example.com' }
    })
    fireEvent.click(screen.getByText('Submit'))

    expect(onSubmit).toHaveBeenCalledWith({
      name: 'Alice',
      email: 'alice@example.com'
    })
  })

  it('名字为空时不提交', () => {
    const onSubmit = jest.fn()
    render(<UserForm onSubmit={onSubmit} />)

    fireEvent.change(screen.getByPlaceholderText('Email'), {
      target: { value: 'alice@example.com' }
    })
    fireEvent.click(screen.getByText('Submit'))

    expect(onSubmit).not.toHaveBeenCalled()
  })

  it('输入更新显示', () => {
    render(<UserForm onSubmit={jest.fn()} />)

    const nameInput = screen.getByPlaceholderText('Name')
    fireEvent.change(nameInput, { target: { value: 'Bob' } })

    expect(nameInput).toHaveValue('Bob')
  })
})
```

### 案例 3：测试 Python 函数

**函数代码：**

```python
def calculate_discount(price, customer_type, is_member):
    """
    计算折扣价格
    - VIP: 8折
    - 普通会员: 9折
    - 非会员: 无折扣
    """
    if price < 0:
        raise ValueError("Price cannot be negative")
    
    if customer_type == "VIP":
        return price * 0.8
    elif customer_type == "regular" and is_member:
        return price * 0.9
    else:
        return price
```

**生成 pytest 测试：**

```
用 pytest 为这个折扣计算函数写测试，覆盖：
- 各种客户类型
- 会员/非会员
- 负数价格抛异常
- 边界值（价格为0）
```

**生成的测试：**

```python
import pytest
from discount import calculate_discount

def test_vip_discount():
    assert calculate_discount(100, "VIP", False) == 80
    assert calculate_discount(100, "VIP", True) == 80

def test_regular_member_discount():
    assert calculate_discount(100, "regular", True) == 90

def test_non_member_no_discount():
    assert calculate_discount(100, "regular", False) == 100
    assert calculate_discount(100, "new", False) == 100

def test_negative_price_raises():
    with pytest.raises(ValueError, match="Price cannot be negative"):
        calculate_discount(-10, "VIP", True)

def test_zero_price():
    assert calculate_discount(0, "VIP", True) == 0
    assert calculate_discount(0, "regular", False) == 0

@pytest.mark.parametrize("price,expected", [
    (50, 40),
    (100, 80),
    (200, 160),
])
def test_vip_various_prices(price, expected):
    assert calculate_discount(price, "VIP", True) == expected
```
